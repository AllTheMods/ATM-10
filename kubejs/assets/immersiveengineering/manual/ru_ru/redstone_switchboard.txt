Коммутатор красного камня
Осмысленные соединения
При работе с несколькими сетями проводов красного камня, соединение отдельных цветов между собой может занимать много места.<br>
Коммутатор решает эту проблему, позволяя передавать любые сигналы из одной сети в другую с возможностью переназначения цветов в процессе.<br>
Сигналы передаются только в одном направлении: слева направо, если стоять перед коммутатором.<br>
Дополнительно коммутатор содержит аналоговые инверторы для всех сигналов, позволяя передавать в выходную сеть инвертированный входной сигнал.<np>
Для настройки коммутатора используйте <link;engineers_screwdriver;отвёртку инженера> чтобы открыть интерфейс. Нажмите на любой входной разъём чтобы начать подключение кабеля, затем нажмите на выходной разъём чтобы завершить соединение.<br>
При необходимости можно переключить красную кнопку "инвертировать" на любой из выходных панелей - учтите, что инверсия работает только при наличии соединения с этим выходом.

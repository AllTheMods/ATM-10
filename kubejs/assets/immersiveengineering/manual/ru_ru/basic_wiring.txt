<&images>Электросети включают три ключевых элемента: §lконнекторы§r для ввода/вывода энергии, §lреле§r для соединения проводов и <link;transformers;трансформаторы>, преобразующие напряжение.<np>
<&connector_recipes>§lКоннекторы§r служат точками подключения к сети. Подключённые устройства будут передавать энергию через сеть.<br>§lРеле§r не передают энергию, а выступают узлами сети.<np>
<&wirecoil_recipes>Для подключения используйте §lкатушку провода§r: ПКМ по первому устройству, затем по второму. Соединение разорвётся при наличии препятствия.<br>Скорость передачи зависит от самого слабого участка цепи.<np>
Коннекторы и реле работают только с проводом своего напряжения. Трансформаторы принимают два провода разного напряжения.<br>
Низко- и средневольтные провода можно изолировать для безопасности.<br>
Shift+ПКМ по коннектору выберет подключённый провод из инвентаря.<np>
Коннекторы имеют лимит передачи (§o<config;iA;machines.wireConnectorInput>§r) IF/t, а провода — предельную нагрузку (§o<config;iA;wires.wireTransferRate;l3>§r) IF/t.<br>
Несколько коннекторов могут §oобъединяться§r через реле в один провод. При превышении лимита §oпровод перегорит§r.<np>
Энергия передаётся между рабочими коннекторами, что позволяет использовать высоковольтные линии для дальних перебросок энергии.

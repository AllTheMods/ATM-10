Компоненты для крафта
Неотъемлемые части
Многие машины и инструменты состоят из различных компонентов. Некоторые из них вам придётся использовать регулярно, как описано далее.<br>
Простые компоненты можно создать в обычном §2Верстаке§r, но для сложных потребуется <link;engineers_workbench;специализированный верстак>.<np>
<&basic_engineering>§2Базовый инженерный блок§r — основной строительный материал для многих машин. Формирует каркас большинства одноблочных механизмов, особенно в системах транспортировки и хранения энергии.
<&mech_components>§2Железные§r и §2стальные механические компоненты§r используются в большинстве машин.<br>
Поскольку они требуются часто, рекомендуется создать для них <link;blueprints;Чертёж>.<np>
<&component_blueprint>
Инструкции по созданию чертежа можно найти <link;blueprints;здесь;molds_components>.<br>
Использование чертежа в <link;engineers_workbench;Верстаке инженера> позволяет создавать простые компоненты с меньшими затратами.<br>
Для продвинутых компонентов чертежи обязательны (см. далее).<np>
<&elec_components>§2Электронные компоненты§r сложнее и используются реже механических. Необходимы для устройств с расширенным управлением, таких как <link;item_batcher;Сортировщик предметов> или <link;turrets;Турели>.<np>
<&light_bulbs>§2Лампы накаливания§r с угольной нитью из бамбука/бумаги используются для создания <link;engineered_lighting;электрических фонарей;electric_lantern>.<np>
<&tubes>§2Электронные лампы§r — дорогие компоненты для <link;engineers_circuit_workbench;Логических схем> и сложной электроники.<np>
<&pcb>§2Монтажная плата§r — изолированная медная основа для пайки <link;engineers_circuit_workbench;Логических схем>.

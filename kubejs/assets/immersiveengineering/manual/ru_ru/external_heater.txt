Внешний нагреватель
Теперь ты готовишь на флюксе!
<&recipe>Внешний нагреватель преобразует флюкс в тепло, передаваемое через боковые вентиляционные отверстия.<br>
Не может создавать:<br>
- Атмосферу для доменных печей<br>
- Дым для копчения пищи<br>
- Температуру для обжиговых печей<np>
Рядом с обычной печью обеспечивает нагрев без расхода топлива.<br>
Настройка сторон Инженерным молотком (Shift для противоположной стороны).<br>
Потребление:<br>
- 8 флюкс/t на печь в рабочем режиме<br>
- 32 флюкс/t при максимальном нагреве (ускорение плавки)<br>
- ×4 при разогреве до рабочей температуры<br>
Редстоун-сигнал отключает ускорение.<br>
Активируется только для печей с предметами и свободным местом для выплавки.

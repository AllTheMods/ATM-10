Интерфейс машины  
Быстрый контроль сложных условий  

<&recipe>С ростом сложности многоблочных машин отслеживание их состояния становится трудной задачей.<br>  
Хотя <link;redstone_probe;компараторные сигналы> могут показывать количество предметов внутри, только Интерфейс машины способен предоставить детальную информацию и отправлять редстоун-сигналы.<np>  

При установке на управляющую редстоун-поверхность многоблочной машины, через GUI можно настроить различные условия для отправки сигналов.<br>  
Нажмите кнопку "Добавить условие" для создания новой строки. В строке: первая кнопка удаляет её, вторая настраивает цвет выходного сигнала.<np>  

Далее идут два выпадающих меню:  
1) Выбор типа проверки ("Активность машины", "Ввод предметов", "Вывод жидкости")  
2) Выбор условия проверки  

Большинство проверок поддерживают "Выход компаратора" - сигнал, пропорциональный количеству предметов/жидкости/энергии. Также доступны логические условия ("Заполнено на 75%", "Пусто").<np>  

При выполнении условия Интерфейс машины отправит соответствующий сигнал через настроенный цветовой канал на подключённый <link;redstone_wiring;Редстоун интерфейсный разъём;bundled>.<br>  

Интерфейс также может принимать редстоун-сигналы и передавать их машине. Цвет сигнала настраивается в правом нижнем углу GUI.<np>  

Это позволяет использовать <link;logic_unit;Логический модуль> для анализа сигналов от машины и отправки управляющих сигналов обратно!

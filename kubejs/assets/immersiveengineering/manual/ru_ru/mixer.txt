Смеситель  
Чан с тестом для печенья!

Смеситель — это многоблочная структура для растворения твёрдых веществ в жидкостях и их смешивания. Конструкция собирается, как показано выше, и активируется с помощью Инженерного молотка по центральному блоку из листового металла.<np>

Наиболее распространённые применения этой машины:
- Приготовление <link;concrete;Жидкого бетона;liquid>
- Создание <link;industrial_fluids;Жидких зелий;potion> из воды и различных ингредиентов<br>

Жидкости подаются в смеситель через порт в нижней части блока питания. Предметы можно загружать через два люка сзади. Передний жидкостный порт служит для вывода продукции.<br>

Подача редстоун-сигнала на переднюю панель управления останавливает работу машины. Это поведение можно изменить с помощью Инженерной отвёртки.

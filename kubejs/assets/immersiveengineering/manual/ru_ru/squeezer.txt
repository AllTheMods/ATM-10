Пресс
Под давлением

Пресс - это многоблочная конструкция для отжима соков и масел из органических материалов при помощи мощного поршня. Конструкция собирается по указанной схеме и активируется с помощью Инженерного Молотка по центральной деревянной бочке со стороны redstone-порта.<br>
Предметы загружаются через два люка сзади, отмеченные синими точками. Люк с оранжевой точкой служит для выхода предметов, а блок под ним - для выхода жидкостей.<br>
Подача redstone-сигнала на переднюю панель управления останавливает работу машины. Это поведение можно изменить с помощью Инженерной Отвёртки.<np>

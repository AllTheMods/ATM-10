Радиовышка  
Длинноволновая радиостанция REM-64  

Хотя для передачи энергии на большие расстояния требуются <link;basic_wiring;высоковольтные линии>, информацию можно передавать беспроводным способом.<np>  

§lРадиовышка§r способна транслировать редстоун-сигналы для приёма другими вышками.<br>  
Для взаимодействия две вышки должны быть настроены на одну частоту и находиться в зоне действия друг друга.<br>  
Дальность сигнала значительно снижается, если на пути есть сооружения или рельеф выше самой вышки. Для максимального покрытия размещайте вышки на открытой местности или вершинах гор/зданий.<np>  

§lКонструкция:§r  
Собранная вышка имеет три порта на верхней части основного блока:<br>  
- §2Левый порт§r — для подачи энергии (потребление: <config;i;machines.radio_tower_consumption> Flux/t)<br>  
- §2Центральный порт§r — для передачи/приёма сигналов через <link;redstone_wiring;редстоун интерфейсный разъём;bundled><br>  
- §2Правый порт§r — для дистанционного управления (выбор 1 из 16 сохранённых частот через цветной сигнал)<np>  

§lИнтерфейс:§r  
- Регулятор частоты (128-384 кГц)  
- 16 цветных кнопок для сохранения частот  
- Дисплей справа показывает другие вышки в радиусе действия  
- Поле внизу отображает текущий радиус покрытия (в метрах)

Сортировщик предметов
Собираем всё вместе
<&recipe>Сортировщик предметов собирает наборы предметов с конвейера перед отправкой, оптимизируя размеры партий для машин.<np>

Режимы работы:
• §2Пакетный режим§r - предметы накапливаются, пока не будут удовлетворены все фильтры
• §2Одиночный режим§r - предметы выводятся сразу при достижении нужного количества

Особенности:
• Нижние соединения задают цвет <link;redstone_wiring;редстоун-провода> через <link;redstone_wiring;Интерфейсные коннекторы;bundled>
• При выполнении фильтра подаётся сигнал по заданному цвету
• Если несколько фильтров используют один цвет, сигнал появится только при выполнении всех
• Сигналы можно использовать для автоматизации фабрик
